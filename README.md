This is a chat written using technologies such as Node.js, Express, Socket.io

## Installation
1. Cloning this repo;
2. Run `npm install` in necessary folder.

## Run Application
1. Run `npm start`;
2. Go into `localhost:8000`(`127.0.0.1:8000`)
